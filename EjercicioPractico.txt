2. Describa los cuatro atributos importantes que debe tener todo producto de software. Sugiera además cuatro atributos adicionales que podrían ser significativos para productos específicos.
Mantenibilidad: El software debe de estar estructurado de forma tal que este pueda evolucionar para poder satisfacer las necesidades y gusto del cliente. Esto debe incluir corregir errores, mejorar el rendimiento a adaptarlo a un entorno nuevo.
Confiabilidad: El software debe ser confiable, es decir, en caso de fallas o errores que estos no cuesten pérdidas económicas o físicas.
Eficiencia: El software debe de hacer el uso adecuado de los recursos del sistema, de forma que se aprovechen al máximo cada uno de los componentes y herramientas.
Aceptabilidad: El software debe de ser aceptable para el usuario, es decir, debe cumplir con los requisitos del cliente además de ser compatible con otros sistemas operativos del cliente
